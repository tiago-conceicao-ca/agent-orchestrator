---
title: Dashboard notifier
description: Retain CAHI notifications in the local web dashboard.
---

The dashboard notifier records CAHI notifications in a local JSONL store and streams
the latest retained records to the web dashboard over the existing local mux
WebSocket.

```yaml
notifiers:
  dashboard:
    plugin: dashboard
    limit: 50

notificationRouting:
  urgent: [dashboard]
  action: [dashboard]
```

## Setup

```bash
cahi setup dashboard
cahi setup dashboard --limit 100 --routing-preset all
cahi setup dashboard --status
```

`limit` controls how many recent notifications are retained for browser reloads
and dashboard restarts. The default is `50`.

## Test

```bash
cahi notify test --to dashboard --template basic
```
